# ⚡ SpeedWifi

Una página simple para medir la velocidad de internet en tiempo real.  

## 🌐 Ver online

[https://itznobodyy.github.io/SpeedWifi/](https://ikecxyy.github.io/SpeedWifi/)

## 🚀 Cómo usar

1. Abre `index.html` en tu navegador.
2. La prueba empieza sola.
3. Usa el botón de pausa para detener y el de play para reiniciar.
